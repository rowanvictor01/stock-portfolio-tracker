# Stock Portfolio Tracker

A CLI application that lets users track a stock portfolio, save it to a file, and load it back later.

## Core Features

[ ] Add stocks to portfolio
[ ] Fetch stock prices from an API
[ ] Calculate profit/loss for each stock and total portfolio value
[ ] Display portfolio in a formatted table with color coding (green for gains, red for losses)
[ ] Save portfolio data to a JSON file
[ ] Load portfolio from file when program starts
[ ] Menu-driven interface
